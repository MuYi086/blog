## 解决win10扩展出来的屏幕模糊

#### 背景
使用 `ghost` 还原系统后, 发现扩展出来的第二屏显示模糊

#### 解决办法
设置 => 显示 => 切换到第二块显示器
将缩放与布局设置为 `100%` ；将显示分辨率调整到最高
