常用Markdown数学公式语法 ​

行内与独行 ​

  1. 行内 $xyz$ => $xyz$
  2. 独行 $$xyz$$ => $$xyz$$

上标、下标与组合 ​

  1. 上标符号 $x^4$ => $x^4$
  2. 下标符号 $x_4$ => $x_4$
  3. 组合符号 $x^{-rft}$ => $x^{-rft}$

汉字、字体与格式 ​

  1. 下划线 $\underline{x+y}$ => $\underline{x+y}$
  2. 上大括号 $\overbrace{a+b+c+d}^{2.0}$ => $\overbrace{a+b+c+d}^{2.0}$
  3. 下大括号 $a+\underbrace{b+c}_{1.0}+d$ => $a+\underbrace{b+c}_{1.0}+d$
  4. 上位符号 $\vec{x}\stackrel{\mathrm{def}}{=}{x_1,\dots,x_n}$ => $\vec{x}\stackrel{\mathrm{def}}{=}{x_1,\dots,x_n}$

占位符 ​

  1. 空格符号 $x \quad y$ => $x \quad y$
  2. 俩个空格符号 $x \qquad y$ => $x \qquad y$
  3. 大空格 $x \ y$ => $x \ y$
  4. 中空格 $x : y$ => $x : y$
  5. 小空格 $x , y$ => $x , y$
  6. 紧贴 $x!y$ => $x!y$

定界符号与组合 ​

  1. 括号 $（）\big(\big) \Big(\Big) \bigg(\bigg) \Bigg(\Bigg)$ => $（）\big(\big) \Big(\Big) \bigg(\bigg) \Bigg(\Bigg)$

  2. 中括号 $[x+y]$ => $[x+y]$

  3. 大括号 ${x+y}$ => ${x+y}$

  4. 自适应括号 $\left(x\right)$，$\left(x{yz}\right)$ => $\left(x\right)$，$\left(x{yz}\right)$

  5. 组合公式 ${n+1 \choose k}={n \choose k}+{n \choose k-1}$ => ${n+1 \choose k}={n \choose k}+{n \choose k-1}$

  6. 组合公式 $\sum_{k_0,k_1,\ldots>0 \atop k_0+k_1+\cdots=n}A_{k_0}A_{k_1}\cdots$ => $\sum_{k_0,k_1,\ldots>0 \atop k_0+k_1+\cdots=n}A_{k_0}A_{k_1}\cdots$

四则运算 ​

  1. 加法运算 $x+y=z$ => $x+y=z$
  2. 减法运算 $x-y=z$ => $x-y=z$
  3. 加减运算 $x \pm y=z$ => $x \pm y=z$
  4. 减加运算 $x \mp y=z$ => $x \mp y=z$
  5. 乘法运算 $x \times y=z$ => $x \times y=z$
  6. 点乘运算 $x \cdot y=z$ => $x \cdot y=z$
  7. 星乘运算 $x \ast y=z$ => $x \ast y=z$
  8. 除法运算 $x \div y=z$ => $x \div y=z$
  9. 斜法运算 $x/y=z$ => $x/y=z$
  10. 分式表示 $\frac{x+y}{y+z}$ => $\frac{x+y}{y+z}$
  11. 分式表示 ${x+y} \over {y+z}$ => ${x+y} \over {y+z}$
  12. 绝对值表示 $|x+y|$ => $|x+y|$

高级运算 ​

  1. 平均数运算 $\overline{xyz}$ => $\overline{xyz}$
  2. 平方根运算 $\sqrt x$ => $\sqrt x$
  3. 开方运算 $\sqrt[3]{x+y}$ => $\sqrt[3]{x+y}$
  4. 对数运算 $\log(x)$ => $\log(x)$
  5. 极限运算 $\lim^{x \to \infty}_{y \to 0}{\frac{x}{y}}$ => $\lim^{x \to \infty}_{y \to 0}{\frac{x}{y}}$
  6. 极限运算 $\displaystyle \lim^{x \to \infty}_{y \to 0}{\frac{x}{y}}$ => $\displaystyle \lim^{x \to \infty}_{y \to 0}{\frac{x}{y}}$
  7. 求和运算 $\sum^{x \to \infty}_{y \to 0}{\frac{x}{y}}$ => $\sum^{x \to \infty}_{y \to 0}{\frac{x}{y}}$
  8. 求和运算 $\displaystyle \sum^{x \to \infty}_{y \to 0}{\frac{x}{y}}$ => $\displaystyle \sum^{x \to \infty}_{y \to 0}{\frac{x}{y}}$
  9. 积分运算 $\int^{\infty}_{0}{xdx}$ => $\int^{\infty}_{0}{xdx}$
  10. 积分运算 $\displaystyle \int^{\infty}_{0}{xdx}$ => $\displaystyle \int^{\infty}_{0}{xdx}$
  11. 微分运算 $\frac{\partial x}{\partial y}$ => $\frac{\partial x}{\partial y}$

逻辑运算 ​

  1. 大于等于运算 $x+y \geq z$ => $x+y \geq z$
  2. 小于等于运算 $x+y \leq z$ => $x+y \leq z$
  3. 不等于运算 $x+y \neq z$ => $x+y \neq z$
  4. 不大于等于 $x+y \ngeq z$ => $x+y \ngeq z$
  5. 不大于等于 $x+y \not\geq z$ => $x+y \not\geq z$
  6. 不小于等于 $x+y \nleq z$ => $x+y \nleq z$
  7. 不小于等于 $x+y \not\leq z$ => $x+y \not\leq z$
  8. 约等于运算 $x+y \approx z$ => $x+y \approx z$
  9. 恒定等于运算 $x+y \equiv z$ => $x+y \equiv z$

集合运算 ​

  1. 属于运算 $x \in y$ => $x \in y$
  2. 不属于运算 $x \notin y$ => $x \notin y$
  3. 不属于运算 $x \not\in y$ => $x \not\in y$
  4. 子集运算 $x \subset y$ => $x \subset y$
  5. 子集运算 $x \supset y$ => $x \supset y$
  6. 真子集运算 $x \subseteq y$ => $x \subseteq y$
  7. 非真子集运算 $x \subsetneq y$ => $x \subsetneq y$
  8. 真子集运算 $x \supseteq y$ => $x \supseteq y$
  9. 非真子集运算 $x \supsetneq y$ => $x \supsetneq y$
  10. 非子集运算 $x \not\subset y$ => $x \not\subset y$
  11. 并集运算 $x \cup y$ => $x \cup y$
  12. 交集运算 $x \cap y$ => $x \cap y$
  13. 差集运算 $x \setminus y$ => $x \setminus y$
  14. 同或运算 $x \bigodot y$ => $x \bigodot y$
  15. 同与运算 $x \bigotimes y$ => $x \bigotimes y$
  16. 实数集合 \mathbb{R} => $\mathbb{R}$
  17. 自然数集合 \mathbb{Z} => $\mathbb{Z}$
  18. 空集 $\emptyset$ => $\emptyset$

  1. 无穷 $\emptyset$ => $\emptyset$
  2. 虚数 $\imath$ => $\imath$
  3. 虚数 $\jmath$ => $\jmath$
  4. 数学符号 $\hat{a}$ => $\hat{a}$
  5. 数学符号 $\check{a}$ => $\check{a}$
  6. 数学符号 $\breve{a}$ => $\breve{a}$
  7. 数学符号 $\tilde{a}$ => $\tilde{a}$
  8. 数学符号 $\bar{a}$ => $\bar{a}$
  9. 矢量符号 $\vec{a}$ => $\vec{a}$
  10. 数学符号 $\acute{a}$ => $\acute{a}$
  11. 数学符号 $\grave{a}$ => $\grave{a}$
  12. 数学符号 $\mathring{a}$ => $\mathring{a}$
  13. 一阶导数符号 $\dot{a}$ => $\dot{a}$
  14. 二阶导数符号 $\ddot{a}$ => $\ddot{a}$
  15. 上箭头 $\uparrow$ => $\uparrow$
  16. 上箭头 $\Uparrow$ => $\Uparrow$
