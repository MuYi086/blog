## 3种常用且稳定的梯子

## 前言
增加了心心念念的 `v2ray` 部分。在这之前服务器环境都已搭建和配置好,本文仅展示梯子的使用，且不讨论 `tor` , `xx-net` 等其他方式。

## 方法一: v2ray
一款优秀的代理集成工具。这里使用了 `V2ray + WebSocket + TLS + Nginx + Cloudflare` 搭建
* 优点 :  高匿,快速, 支持多平台, 多协议
* 缺点 : 配置较复杂,上手需要结合多种技术
