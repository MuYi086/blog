代码规范 ​

  1. 样式文件必须写上 @charset 规则，并且一定要在样式文件的第一行首个字符位置开始写，编码名用 UTF-8

  2. 字符 @charset ""; 都用小写字母，不能出现转义符，编码名允许大小混写

  3. 考虑到在使用 "UTF-8" 编码情况下 BOM 对代码的污染和编码显示的问题，在可控范围下，坚决不使用 BOM。

    css
    @charset "UTF-8";

.jdc{}
