# 科学上网-XX-NET

## 介绍

::: warning 注意
最新项目处于半开源状态，已开始收费订阅
:::

这个项目使用的 `gae(google app engine)` ,需要自行注册 `google` 账号并开启 `gae` 服务.每个 `app` 一天1个G流量，最大上限12个 `app` ，即最多每天12G流量
