### GitBook安装和常用命令
##### 安装GitBook
```
>_ npm install gitbook-cli -g
```

查看是否安装成功:
```
>_ gitbook -v
```

##### 初始化GitBook
```
>_ gitbook init
```

##### 配置文件install
```
>_ gitbook install ./
```

##### 预览
```
>_ gitbook serve
```

预览页面`http://localhost:4000`

##### 编译GitBook
```
>_ gitbook build
```
