GitBook安装和常用命令

安装GitBook
npm install gitbook-cli -g

查看是否安装成功:

gitbook -v
初始化GitBook
gitbook init
配置文件install
gitbook install ./
预览
gitbook serve

预览页面http://localhost:4000

编译GitBook
gitbook build
